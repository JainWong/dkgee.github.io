# 信管国考题分析

这里列举了对人大信管专业的历年(2004-2019)国考题综合分析结果，供参考学习。

各科整理的答案

【信息资源管理】 【图书管学】 【情报学】 【档案学】

建议

每个课程的连续考试题如名词1、2、3……30，按照A、B、C划分为三个部分，每部分包含1~10道题，对每个段进行背诵复习，方便记忆。如果困难，可以依照此方法对某小段再分。

  1. 历年国考题：以其为基础。
  2. 选专业：选择自己要考的专业（如信息资源管理+情报学）。选择后，国考的论述题就只背这两个专业的，一定要多看多背。
  3. 三三段式：对名词、简答、论述，按此方法划分。
  4. 背诵技巧：信息资源管理、图书、情报、档案的名词、简答题全背；论述题只背选择专业部分（建议不要背其他科的）。摘抄、背诵、思考。

快速浏览技巧（IOS系统为例）

  1. 打开页面后，例如信管。会看到整个文档链接目录。
  2. 先分区背诵考题，如果想浏览某个考题的答案，以 “社会信息化”为例，手指在屏幕上按住该名词。浏览器会弹窗预览，自动定位到答案处。
  3. 也可直接点击链接，自动跳到到答案处。
