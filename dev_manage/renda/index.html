# 人大在职研考试分析

通过自己对人大在职研考试的经验技巧分析，帮助自己能够尽快通过人大的在职研究生考试。

鹅鹅鹅

谁愿意考研，谁愿意学习，有病啊，都是被发展形式所逼的！！！

# 期末课程考试

课程已经修完了，考试也基本通过了。从18年后，在职研的日常课程考试由课后随堂考试，改到期末统一考试。间接变得严格了。

# 题库考

必须得交钱获取考试资格卡，才能参加人大题库考试。一门如果一次考不过，下次再考就得再交200块。 我就信息资源管理、信息检索两门专业课，重考了两次。

不过，在两年内把题库都考过了，基本都是六十多分，及格分数。

# 国考

这个就比较难的，四门专业课，全国统一考试，完全是题海战术，就看你持久记忆力了，这个必需得做出充分准备，否则，光靠临时抱佛脚是压根通不过的。

报考了两次，一次是18年没经验去考了18分 😭。第二次是19年报名了，正好自己所在公司当时有事，没去考，更准确的理解是没敢去考 😓。

这个有经验了之后，发现把该专业历年的考试题都背过的话，通过率还是挺高的，自己预估了下，大概重复率分数在70~80分之间。基本通过。
